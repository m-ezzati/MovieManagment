# Cinema Management Project

## Overview
The **Movie Management Project** is a Java EE web application for managing movies, showtimes, user's watchlist.  
It uses Servlets, JSP, and Hibernate, and runs on the Apache Tomcat server.

## Features
- Manage movies, showtimes.  
- User and admin panels  
- Database connection using Hibernate ORM  
- JSP pages for dynamic web content  
- Deployment on Apache Tomcat

## Technologies Used
- **Java EE (Servlet, JSP)**
- **Hibernate**
- **PostgreSQL**
- **HTML / CSS / JS**
- **Apache Tomcat**
- **IntelliJ IDEA**

## Requirements
Before running the project, make sure you have:
- **JDK 11** or higher installed  
- **Apache Tomcat 9** or later  
- **MySQL Server** running  
- **Maven** (if used for dependency management)
